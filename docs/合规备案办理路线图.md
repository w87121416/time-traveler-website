# 时光旅人合规备案办理路线图

日期：2026-07-17
主体：南京形而不器科技有限公司
当前状态：官网静态公开预览；无注册、无真实 AI 对话、无支付、无公众下载

## 一、建议办理顺序

### 1. 确定中国大陆正式托管环境

当前 GitHub Pages 可继续作为代码源和过渡预览站，但不能直接作为阿里云 ICP 备案的中国大陆接入主机。通过阿里云申请备案，需要关联阿里云中国内地可备案服务器或符合备案条件的托管资源。

- 我们可以：比较符合静态官网需求的大陆托管方案、准备迁移配置、保留 GitHub 自动构建。
- 公司必须：选择并购买资源；任何下单和付款都由公司审核后完成。
- 官方参考：[阿里云 ICP 备案流程](https://help.aliyun.com/zh/icp-filing/basic-icp-service/user-guide/icp-filing-application-overview)

### 2. 非经营性 ICP 备案

当前仅做免费产品介绍，可先按非经营性互联网信息服务准备 ICP 备案。后续若提供有偿 AI 服务、会员、广告或被认定为网络游戏，再单独评估 ICP 许可证等经营资质。

准备材料：

- 营业执照或电子营业执照。
- 法定代表人和网站负责人证件、手机号、邮箱。
- 域名实名认证证明。
- 中国大陆服务器和接入信息。
- 网站名称、服务内容、域名/IP、负责人授权书（如适用）。

我们可以整理字段、检查材料和在阿里云后台协助填写；实名认证、人脸/短信核验、承诺确认、签章及最终提交必须由公司负责人完成。

官方依据：[《非经营性互联网信息服务备案管理办法》](https://www.miit.gov.cn/gyhxxhb/jgsj/cyzcyfgs/bmgz/xxtxl/art/2024/art_84a0cfa0ebd049bbbe751dca9a008e56.html)

### 3. 公安联网备案

网站正式联网后，应按要求在三十日内办理公安联网备案。建议记录首次公开日期，并尽快向南京属地公安网安部门确认当前境外过渡托管站点的办理口径。

我们可以准备域名/IP、接入商、服务类型、网络安全管理制度、应急预案和负责人信息；实名注册、承诺和最终提交由公司完成。

办理入口：[全国公安机关互联网站安全管理服务平台](https://beian.mps.gov.cn/)

### 4. 拟人化 AI 服务安全评估

时光旅人的持续陪伴、角色扮演和 AI 对话具有明显拟人化互动特征。首次开放真实 AI 服务或新增相关功能前，应向运营主体所在地省级网信部门提交安全评估报告。

我们可以起草：

- 安全评估报告及证据目录。
- 极端情绪、自伤风险、诈骗和重大财产损失干预流程。
- 未成年人和老年人保护方案。
- 沉迷与依赖提醒、连续使用两小时提醒机制。
- 投诉举报、人工申诉和应急响应制度。
- 测试用例、结果记录和整改模板。

技术负责人必须确认材料与真实系统一致，法定代表人或授权负责人负责签章和提交。

官方依据：[《人工智能拟人化互动服务管理暂行办法》](https://www.cac.gov.cn/2026-04/10/c_1777558395078289.htm)

### 5. 算法备案

公开 AI 服务前准备算法备案，形成算法名称、应用场景、基本原理、目的与主要机制、数据和模型说明、安全机制及算法自评估报告。取得备案编号后，在产品显著位置公示编号并链接官方公示页面。

我们可以完成文字初稿、对外公示稿和备案系统字段预填；研发确认技术真实性，公司备案账号持有人完成企业认证、承诺和提交。

办理入口：[互联网信息服务算法备案系统](https://beian.cac.gov.cn/)

### 6. 生成式 AI 模型备案或应用登记

先确定真实技术路线：

- 自研、自行部署或以公司名义提供经过实质性优化的模型：向属地网信部门咨询生成式 AI 服务备案。
- 直接调用已经备案的大模型 API：通常准备应用或功能登记，并取得上线编号。

模型权重、Prompt、API 密钥和商业合同不需要公开；但上线后的模型名称、备案号或上线编号不能完全隐藏。

官方参考：[国家网信办生成式人工智能服务备案公告](https://www.cac.gov.cn/2026-05/13/c_1780413225190669.htm)

### 7. 游戏属性与版号预咨询

时光旅人包含角色、成长、互动、旅行收集和游戏化 Skill，且计划网上下载并联网 AI 对话，建议按“较高概率需要游戏出版审批”准备，并向江苏省出版主管部门或具有网络游戏出版范围的出版单位取得书面预咨询结论。

如果被认定为国产网络游戏，通常需要出版单位、ICP 相关资质、软件著作权、完整中文脚本、角色与场景资料、安装包、审查账号或激活码、演示视频等材料。出版合作可能涉及合同和费用，必须由公司审核决定。

官方依据：[国产网络游戏审批指南](https://www.nppa.gov.cn/bsfw/xksx/cbfxl/wlcbfwspsx/202210/t20221013_600725.html)

## 二、公开与保密

可以不公开：模型权重、系统 Prompt、过滤阈值、完整训练语料、API 密钥、供应商价格与合同、精确机房/IP、内部评估报告全文。

不能一概隐藏：运营主体和联系方式、真实个人信息处理规则、接收个人信息的第三方、模型名称与备案或上线编号、算法备案编号及基本机制、投诉申诉入口、AI 身份和生成内容标识。

## 三、下一轮办理所需资料

1. 营业执照或电子营业执照，用于复核主体名称、统一社会信用代码和地址。
2. 拟任网站负责人及可用于备案的手机号、邮箱。
3. 阿里云账号中是否已有中国大陆可备案服务器或托管资源。
4. 域名实名认证主体是否为南京形而不器科技有限公司。
5. 真实模型路线已由负责人私下提供；待用生产环境配置、服务地域和合同主体复核，准确型号不写入公开仓库。
6. 软件著作权已申请；待补申请主体、申请日期、申请号或受理号，获证后补登记证书与登记号。
7. 是否同意向江苏省网信、出版主管部门进行上线前书面咨询。

企查查页面可辅助核对，但不能代替营业执照、实名认证、签章或主管部门审核。任何购买、付款、签约和最终提交均由公司审查后进行。
